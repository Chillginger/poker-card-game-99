# poker-card-game-99
player v.s. bot card game.

Rules:
keep the total number 'Sea' around 0~99. 
whoever let 'Sea' become more than 99, lose.

1, 2, 3, 6, 7, 8, 9 can plus the 'Sea'
15(spade Ace) can set the 'Sea' into 0
4, 5, 11 can skip your round
10 can either +10 or -10 the 'Sea'
12 can either +20 or -20 the 'Sea'
13 can set the 'Sea' into 99
